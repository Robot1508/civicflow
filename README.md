# Civicflow: Equitably Routing Civic Infrastructure 🚀

**Live Prototype:** [https://fixithub-5fe6e.web.app](https://fixithub-5fe6e.web.app)

## 🌟 The Vision
FixIt Hub eliminates "Data Deserts" in urban environments. By utilizing **Gemini 1.5 Flash**, we ensure that infrastructure repair is driven by objective need and social equity, specifically targeting underserved areas like **Wards 9 and 11**.

## 🛠️ Tech Stack
- **AI:** Google Gemini 1.5 Flash (Multimodal)
- **Backend:** Firebase Cloud Functions (Gen 2)
- **Database:** Cloud Firestore
- **Frontend:** React + Vite + Tailwind CSS
- **Infrastructure:** Google Cloud Platform (GCP)

## 🚀 Key Features
- **AI Verification:** Real-time image analysis of infrastructure damage.
- **Equity-Boost:** Algorithmic priority for neglected wards to ensure social justice.
- **XAI Transparency:** Plain-language explanations for all AI-driven decisions.

## 🌍 UN SDG Alignment
Supports **SDG 11: Sustainable Cities and Communities** by reducing urban inequality through data-driven infrastructure management.
